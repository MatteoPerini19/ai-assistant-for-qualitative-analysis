






### Qualitative data location 

Here is where to find the qualitative data and how to match them with specific observations.

In the main dataset (.csv file) there are specific variables containing the name of the audio file and/or text file specific to that row of the dataset (e.g., text from participant 19 reflection 2). 
Text file, if present, are located in data/qualitative_data/text_files.
Audio file, if present, are located in data/qualitative_data/audio_files.
